## 💬FireTalk - 실시간 채팅어플
- FireBase를 사용한 실시간 채팅어플

### 개발 기간
- ver1. 2022.03.08 ~ 2022.03.22 
- UI/UX 수정 예정

### 기술 스택 & 라이브러리
- IDE : Android Studio
- Language : Kotlin
- Android Jetpack(dataBinding, ViewBinding, RecyclerView)
- Firebase
  - Authentication
  - Realtime Database
  - Storage

### 기능
- 회원가입 / 로그인 / 구글 로그인 / 로그아웃
- 실시간 채팅
- 프로필 수정 (이미지, 이름 수정 가능)
